# ONNX Runtime Light Glue Example

一个简单的ONNX Runtime Light Glue示例。

测试环境：
- Ubuntu 20.04
- ONNX Runtime onnxruntime-linux-x64-gpu-1.16.3
- CUDA 11.7

![结果](doc/Matches_screenshot_20.01.2024.png)

- 2024-01-20
- 目前只支持super point 和 light glue 的端到端模型
